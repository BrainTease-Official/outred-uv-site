About

I (OutRed) made this website originally back in school for some friends, back when I had no idea how to code. My friends kept on bugging me, so I finally made it. I password protected the website at first so only my friends could access it, and no annoying people. Whenever it got blocked, I would change the URL so my friends could use it again. Eventually, I got ratted out, and had to pay the consequences. The site got blocked, and I didn't want to mess with the firewall, so I took down the password and made the site public.

Ever since then, I have been updating this site as much as possible, with Apps, Unblockers, and Games, trying to make it the best out there. I love working on this, and if you ever need any help or questions, or want to chat in my community, go ahead and join the Discord server.
